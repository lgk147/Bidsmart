## 🎯 Campaign Manager

Create, manage, and optimize all your Amazon advertising campaigns.

### Creating a Campaign

1. Click **"Create Campaign"** button
2. Fill in campaign details:
   - **Campaign Name**: Descriptive name
   - **Campaign Type**: Select ad product (DSP, SP, SB, SBV, SD)
   - **Campaign Objective**: Awareness, Consideration, Conversion, Loyalty
   - **Daily Budget**: Budget per day
   - **Bidding Strategy**: Automatic, Manual, or Dynamic
   - **Start/End Date**: Campaign duration

3. Click **"Create Campaign"**

### Managing Campaigns

**Filter Campaigns:**
- Search by name
- Filter by campaign type
- Filter by status (Active, Paused, Ended)

**Campaign Actions:**
- **Edit** - Modify campaign settings
- **Duplicate** - Create a copy
- **Pause/Resume** - Change campaign status
- **Delete** - Remove campaign

### Campaign Table Columns

- **Campaign Name** - Click to view details
- **Type** - Ad product type
- **Status** - Active, Paused, or Ended
- **Budget** - Total allocated budget
- **Spend** - Amount spent
- **Impressions** - Ad views
- **Clicks** - Click count
- **Conversions** - Purchases
- **ROAS** - Return on ad spend

### Bulk Operations

- **Import** - Upload campaigns from CSV
- **Export** - Download campaign data

## 💰 Bidding Models

AI-powered bid optimization and strategy management.

### Current Performance

Four KPI cards:
- **Avg CPC** - Cost per click
- **Current ROAS** - Return on ad spend
- **Conv. Rate** - Conversion rate
- **Active Bids** - Number of active bids

### Bidding Strategies

Select from four strategies:

1. **Dynamic Bidding** ⭐ Recommended
   - Automatically adjusts bids based on conversion likelihood
   
2. **Fixed Bidding**
   - Set fixed bid amounts for consistent spending
   
3. **Target ROAS** ⭐ Recommended
   - Optimize bids to achieve specific return on ad spend
   
4. **Maximize Conversions**
   - Get the most conversions within budget

### Strategy Parameters

**Auto-Bidding Toggle:**
- Enable/disable automatic bid adjustments

**Target ROAS:**
- Set desired return on ad spend (1.0x - 10.0x)
- Slider for easy adjustment

**Bid Limits:**
- **Max Bid**: Maximum bid amount ($0.50 - $10.00)
- **Min Bid**: Minimum bid amount ($0.10 - $2.00)

**Optimization Settings:**
- Bid Adjustment Mode: Aggressive, Moderate, Conservative
- Optimization Goal: Conversions, ROAS, Reach, Clicks

### Bid Performance Analysis

Charts showing impact of bid levels:
- **ROAS by Bid Amount**
- **Conversions by Bid Amount**
- **Spend by Bid Amount**

### Keyword Bid Recommendations

Table shows:
- Keyword name
- Current bid
- Suggested bid (AI-powered)
- Impressions
- Conversions

**Actions:**
- Apply individual recommendations
- Apply all recommendations at once

### Bid Adjustments

Audience segment modifiers:
- Mobile Devices: +15%
- Desktop: -5%
- Prime Members: +25%
- Repeat Customers: +30%
- High-Income Zip: +20%

### Optimization Score

Circular gauge showing optimization health (0-100):
- 80-100: Excellent
- 60-79: Good
- 40-59: Fair
- 0-39: Needs Improvement

### Saving Strategies

Click **"Save Strategy"** to store configuration, or **"Auto-Optimize"** to let AI optimize.

## 💡 Best Practices

### Campaign Management

✅ **Do:**
- Set clear campaign objectives
- Use descriptive campaign names
- Start with conservative budgets and scale
- Monitor performance daily for first week
- Test multiple ad types

❌ **Don't:**
- Create overlapping campaigns
- Set unrealistic ROAS targets initially
- Pause campaigns too quickly (give 2 weeks min)
- Ignore negative keywords

### Budget Allocation

**Full-Funnel Approach:**
- 40% Awareness (DSP, Prime Video)
- 35% Consideration (SB, SBV)
- 25% Conversion (SP, SD Retargeting)

**Single-Objective Approach:**
- 100% focused on one funnel stage
- Use appropriate ad products

### Bidding Optimization

✅ **Do:**
- Start with target ROAS bidding
- Enable auto-bidding for efficiency
- Apply bid adjustments for high-value segments
- Review keyword bids weekly

❌ **Don't:**
- Set max bids too low
- Change bids too frequently
- Ignore optimization score

### Audience Targeting

✅ **Do:**
- Create specific audience segments
- Use lookalike audiences to scale
- Test different demographic groups
- Retarget cart abandoners

❌ **Don't:**
- Make audiences too broad
- Ignore audience performance data
- Over-segment your audience

### Creative Best Practices

**Images:**
- Use high-quality images (1200x628px minimum)
- Show product clearly
- Include lifestyle shots
- Test multiple creatives

**Copy:**
- Keep headlines under 80 characters
- Include clear call-to-action
- Highlight unique value proposition
- A/B test messaging

**Video:**
- Hook viewers in first 3 seconds
- Keep under 30 seconds for best performance
- Include captions (80% watch without sound)
- Show product in use

### Performance Monitoring

**Daily:**
- Check campaign status
- Monitor budget pacing
- Review top-performing ads

**Weekly:**
- Analyze keyword performance
- Adjust bids
- Review audience segments
- Update budgets

**Monthly:**
- Full performance review
- Strategic planning updates
- Budget reallocation
- Competitive analysis

### Reporting

✅ **Do:**
- Export data regularly for analysis
- Track performance vs. goals
- Document learnings
- Share insights with team

❌ **Don't:**
- Only look at vanity metrics
- Ignore attribution data
- Compare different time periods unfairly

## 🆘 Troubleshooting

### Common Issues

**Problem**: Campaign not spending
- Check campaign status (must be Active)
- Verify bids are competitive
- Ensure budget is adequate
- Check targeting isn't too narrow

**Problem**: High spend, low conversions
- Review audience targeting
- Check landing page experience
- Analyze keyword relevance
- Consider lowering bids

**Problem**: Low impressions
- Increase budget
- Expand targeting
- Raise bids
- Check ad approval status

**Problem**: Data not updating
- Click "Refresh" button
- Check internet connection
- Verify API connection (if applicable)
- Clear browser cache
